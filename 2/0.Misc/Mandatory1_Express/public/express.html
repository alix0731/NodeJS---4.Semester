Express

What is it?

Express is a minimal and flexible Node.js web application framework that provides a robust set of features for web and mobile applications.

To create route for a page you have to use the function get inside app, where you give a string as a argument

Next you use the variables in the callback fuction, where you write res.sendFile(__dirname + "pathToHtmlFile")

"__dirname" is used to located the current location of the folder and then add the folder/file name

We also use app.listen to tell with port the server should run on

Send object with response

We can also send object with response variable

This is how its done:

  • app.get("/", (req, res) => {
    res.send({data = "Ali"});
    });

Other:

  • app.use(express.json()) : This is used when you are converting incomming data to json
  • app.use(express.static('public')) : This is used to make all HTML, CSS & JS file static
